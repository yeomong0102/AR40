디버그 -> 옵션 -> 키보드 후 원하는 단축키 변경

컨트로 A => 코드 전체 선택

컨트롤 시프트 G => #include 파일로 이동

컨트롤 시프트 G => Header <=> Cpp 전환

컨트롤 k f => 코드 탭 줄 정리

컨트롤 K O => cpp 헤더 전환

컨트롤 k C => 주석처리

컨트롤 k U => 주석 제거

컨트롤 delete => 한줄 제거

컨트롤 + 방향키 => 의미있는 곳으로 이동

컨트롤 스페이스 => 자동완성 기능 켜기

end => 줄 끝으로 이동

home => 줄 처음으로 이동

시프트 end => 맨뒤에서 맨 앞까지 선택

시프트 home => 맨 앞에서 맨 뒤까지 선택

컨트롤 + 시프트 + 방향키 => 의미있는 프로그래밍 키워드별 선택

알트 + 시프트 + 방향키 => 다중 라인 선택

컨트롤 U => 소문자로 변경

컨트롤 시프트 U => 대문자로 변경

컨트롤 R R => 연관된 명칭만 변경

F12 타고 들어가기

컨트롤 f 현재 프로젝트 모두 찾기
